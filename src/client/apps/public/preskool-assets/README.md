# Preskool Assets

## 📁 What to Copy Here

Copy your Preskool template's **assets** into the respective subfolders:

### CSS Files
```
Preskool/assets/css/*  →  css/
```
- Main stylesheet (e.g., `style.css`, `main.css`)
- Component CSS files
- Theme CSS files

### JavaScript Files
```
Preskool/assets/js/*  →  js/
```
- Main app.js
- Chart libraries
- Calendar libraries
- Form validation libraries
- Any other JS dependencies

### Images
```
Preskool/assets/images/*  →  images/
```
- Logo files
- Avatar placeholders
- Dashboard icons/images
- UI element images

### Fonts
```
Preskool/assets/fonts/*  →  fonts/
```
- Custom fonts used in template
- Icon fonts (if any)

## 📋 After Copying

Once files are copied, we'll:
1. Update `index.html` to reference these files
2. Extract dashboard components
3. Integrate with your existing API

## ⚠️ Note

Make sure to preserve the folder structure from Preskool if it has subfolders (e.g., `css/lib/`, `js/plugins/`).
