# Preskool Template Files

## 📁 What to Copy Here

Copy your **entire Preskool template** structure into this folder.

This includes:
- All HTML files
- All Angular component files (if any)
- All TypeScript files
- All configuration files
- Documentation files

**Purpose:** This folder serves as a reference for extracting components and understanding the template structure.

## 📋 Example Structure

After copying, this folder should contain something like:

```
preskool-template/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── pages/
│   │   └── ...
│   └── assets/
├── assets/
├── angular.json (or similar)
├── package.json
└── README.md (from Preskool)
```

## ⚠️ Important

- This is a **reference folder** - we won't use these files directly
- We'll extract components and convert them to Angular 20 standalone components
- The actual assets (CSS, JS, images) go to `public/preskool-assets/`
