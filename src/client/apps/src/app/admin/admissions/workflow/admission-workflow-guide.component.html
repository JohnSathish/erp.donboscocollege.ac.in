How this maps to the database

  • Stage 1 — isApplicationSubmitted and isPaymentCompleted (application fee).
  • Stage 2 — Document checks on the application detail; use Submitted while reviewing; Rejected with remarks if documents fail.
  • Stage 3–4 — EntranceExam, WaitingList, Approved, Rejected for merit and decisions.
  • Stage 5–6 — Approved with payment + Enroll action → Enrolled.
